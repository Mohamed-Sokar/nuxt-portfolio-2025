## Code Example

Here’s a small Vue.js snippet that shows a reactive component:

```vue
<template>
  <div>
    <p>{{ message }}</p>
    <button @click="updateMessage">Click Me</button>
  </div>
</template>

<script>
export default {
  data() {
    return {
      message: "Hello, world!",
    };
  },
  methods: {
    updateMessage() {
      this.message = "Updated message!";
    },
  },
};
</script>
```
